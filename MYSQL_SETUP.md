# راه‌اندازی بک‌اند MySQL با XAMPP

این پروژه از این به بعد با MySQL کار می‌کند، نه SQLite.

## 1. سرویس‌های XAMPP
در XAMPP Control Panel این دو سرویس را Start کنید:

- Apache
- MySQL

## 2. مسیر پروژه
فولدر `doping` را داخل مسیر زیر بگذارید:

```txt
C:\xampp\htdocs\doping
```

## 3. تنظیمات دیتابیس
تنظیمات اتصال در این فایل است:

```txt
app/config.php
```

تنظیم پیش‌فرض برای XAMPP:

```php
return [
    'db_host' => '127.0.0.1',
    'db_port' => '3306',
    'db_name' => 'doping_shimi',
    'db_user' => 'root',
    'db_pass' => '',
    'db_charset' => 'utf8mb4',
];
```

در XAMPP معمولاً کاربر `root` بدون رمز است.

## 4. ساخت دیتابیس
لازم نیست دستی جدول‌ها را بسازید. پروژه در اولین اجرا:

1. دیتابیس `doping_shimi` را می‌سازد.
2. جدول‌ها را می‌سازد.
3. داده‌های نمونه، دوره‌ها، مدرس‌ها، و ادمین پیش‌فرض را اضافه می‌کند.

اگر کاربر MySQL اجازه ساخت دیتابیس نداشت، از phpMyAdmin دیتابیس را دستی بسازید:

```txt
doping_shimi
```

Collation پیشنهادی:

```txt
utf8mb4_unicode_ci
```

## 5. آدرس‌های تست

صفحه اصلی داینامیک:

```txt
http://localhost/doping/index.php
```

دوره‌ها:

```txt
http://localhost/doping/courses.php
```

جزئیات دوره:

```txt
http://localhost/doping/course-detail.php?id=1
```

درباره ما:

```txt
http://localhost/doping/about.php
```

پنل پیام دانش‌آموز به مشاور:

```txt
http://localhost/doping/student-panel.php
```

پنل مدیریت:

```txt
http://localhost/doping/admin-login.php
```

## 6. ورود ادمین

```txt
نام کاربری: admin
رمز عبور: admin123
```

## 7. تست امکانات

### انتخاب دوره‌های صفحه اصلی
1. وارد `admin-login.php` شوید.
2. در تب صفحه اصلی، ۳ دوره را انتخاب کنید.
3. ذخیره کنید.
4. `index.php` را رفرش کنید.

### مدیریت دوره‌ها
در تب دوره‌ها می‌توانید:

- دوره جدید اضافه کنید.
- دوره حذف کنید.
- دوره ویرایش کنید.
- پایه، ساعت، استاد، قیمت، تصویر، توضیح کوتاه و کامل را تغییر دهید.
- موارد «در این دوره چه یاد می‌گیری؟» را وارد کنید.
- تیک‌های باکس ثبت‌نام را تغییر دهید.
- نمونه ویدیو و جلسه رایگان را از دیتابیس ویدیوها انتخاب کنید.

### تعداد کاربران دوره
در لیست دوره‌ها فرم «ثبت خرید» وجود دارد. با ثبت خرید، تعداد کاربران/خریداران دوره زیاد می‌شود.

### مدرسین
در تب مدرسین می‌توانید مدرس اضافه کنید. مدرس‌ها در `about.php` نمایش داده می‌شوند و اگر در دوره انتخاب شوند، اطلاعات مدرس در جزئیات دوره خودکار نمایش داده می‌شود.

### ویدیوها
در تب ویدیوها می‌توانید ویدیو اضافه کنید. بعد در فرم دوره، آن ویدیو را به عنوان نمونه ویدیو یا جلسه رایگان انتخاب کنید.

### مشاور
1. از `student-panel.php` پیام ارسال کنید.
2. در پنل مدیریت تب مشاور را باز کنید.
3. پیام را ببینید و پاسخ بدهید.

## 8. خطاهای رایج

### خطای could not find driver
یعنی افزونه `pdo_mysql` فعال نیست. در XAMPP معمولاً فعال است. اگر نبود، در `php.ini` این خط را فعال کنید:

```ini
extension=pdo_mysql
```

بعد Apache را Restart کنید.

### خطای Access denied
اطلاعات `app/config.php` با MySQL شما هماهنگ نیست. نام کاربری/رمز را اصلاح کنید.

### مشکل فارسی
دیتابیس و جدول‌ها با `utf8mb4_unicode_ci` ساخته می‌شوند. اگر دیتابیس را دستی ساختید، همین Collation را انتخاب کنید.

## تغییرات مهم پنل مدیریت

- تعداد کاربران/خریداران هر دوره فقط از رکوردهای پرداخت‌شده محاسبه می‌شود؛ یعنی فقط خریدهایی که در جدول `enrollments` مقدار `paid_amount` بزرگ‌تر از صفر دارند شمرده می‌شوند.
- در پنل مدیریت دوره‌ها، فیلدهای `اسلاگ` و `سطح` از فرم حذف شده‌اند. اسلاگ به صورت داخلی و خودکار از عنوان ساخته می‌شود.
- در بخش «چیزهایی که در دوره یاد می‌گیری»، هر خط باید با `/` جدا شود:

```txt
عنوان / توضیح
```

- در بخش مدرسین، فقط توضیح کامل مدرس دریافت و نمایش داده می‌شود؛ توضیح کوتاه از فرم حذف شده است.

## نصب روی هاست با install.sql

برای هاست‌هایی مثل InfinityFree بهتر است به جای ساخت خودکار دیتابیس توسط PHP، دیتابیس را از پنل هاست بسازید و فایل SQL را ایمپورت کنید.

### مراحل

1. در پنل هاست یک MySQL Database بسازید.
2. وارد phpMyAdmin همان دیتابیس شوید.
3. فایل زیر را Import کنید:

```txt
install.sql
```

4. فایل تنظیمات را باز کنید:

```txt
app/config.php
```

5. اطلاعات دیتابیس هاست را وارد کنید:

```php
return [
    'db_host' => 'HOST_FROM_HOSTING_PANEL',
    'db_port' => '3306',
    'db_name' => 'DATABASE_NAME',
    'db_user' => 'DATABASE_USER',
    'db_pass' => 'DATABASE_PASSWORD',
    'db_charset' => 'utf8mb4',
    'auto_create_database' => false,
    'auto_migrate' => true,
    'auto_seed' => true,
];
```

نکته: در هاست اشتراکی مقدار `auto_create_database` باید `false` بماند چون معمولاً اجازه `CREATE DATABASE` ندارید.

### فایل .htaccess

فایل `.htaccess` اضافه شده و باعث می‌شود سایت به صورت پیش‌فرض از `index.php` بالا بیاید و فایل‌هایی مثل `install.sql` از طریق مرورگر قابل دانلود نباشند.

### بعد از نصب

آدرس پنل مدیریت:

```txt
https://your-domain.com/admin-login.php
```

اطلاعات ورود اولیه:

```txt
admin
admin123
```

بعد از ورود حتماً رمز ادمین را تغییر دهید یا ادمین جدید بسازید.
